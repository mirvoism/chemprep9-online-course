Our 2025 design system follows modern UX principles with chemistry-themed elements, ensuring accessibility, consistency, and engaging user experiences.

### **Secondary Colors (30% - Teal)**

### **Accent Colors (10% - Coral)**
```css

--accent-500: #F76C5E   /* Base coral - used sparingly */
--accent-600: #ea4c3d   /* Hover states */
--accent-900: #962724   /* Deep coral */
```

### **Updated Button Color Strategy**
- **Primary Actions**: `bg-primary-600` (Navy blue) - High contrast, professional
- **Secondary Actions**: `bg-secondary-500` (Teal) - Supporting interactions  
- **Accent Elements**: `accent-500` (Coral) - Reserved for special highlights only

### **Critical Button Visibility Fix**
When Tailwind classes fail to render properly, use inline styles as fallback:
```tsx
// Guaranteed visibility approach
style={{ 
  backgroundColor: '#1e4d7a', // Navy blue
  color: '#ffffff',           // White text  
  border: 'none'
}}
```
